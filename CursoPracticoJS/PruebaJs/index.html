¿Qué es una variable y para qué sirve?

Una variable es una espacio en memoria que permite almacenar un valor y esta sirve para ser utilizada en nuestro codigo.

¿Cuál es la diferencia entre declarar e inicializar una variable?

Declarar es dejar un espacio en memoria con n nombre y inicializar la variable es asignarle un valor a dicha declaracion

¿Qué es una función?

Una funcion es el encasulapmiento de bloques de codigo para reutilizarlos y luego ejecutarla en algun momento al llamarla

¿Cuándo me sirve usar una función en mi código?

Una funcion es util para realizar una tarea cuando ya tengamos nuestras variables y poder realizar tareas con estos , ademas cuando esten muy repetidas y se puedan encapsular y reutilizarlos

¿Cuál es la diferencia entre parámetros y argumentos de una función?

Un parametro son aquellas datos que recibe una funcion cuando se estan creando y argumento son aquellos que se le asignan cuando se llama una funcion

¿Qué es un condicional?

Un condicional es una operacion que comprueba si algo se cumple o no , trabaja con valores de true y false .

¿Qué tipos de condicionales existen en JavaScript y cuáles son sus diferencias?

Existe el if , else if , else : el If es para decir que pasa si se cumple cierta condicion , el else para es el resultado de cuando no se cumple la condicion y else if , es otra condicion que tiene que comprobar si la anterior no se cumple

¿Puedo combinar funciones y condicionales?

Por Supuesto , podemos combinarlas para realizar tareas de acuerdo a una condicion y en base a esto generar un resultado

¿Qué es un ciclo?

Un Ciclo es un bucle o proceso que se repite mientras se cumplan ciertos parametros o condiciones

¿Qué tipos de ciclos existen en JavaScript?

For , While , switch

¿Qué es un ciclo infinito y por qué es un problema?

Un ciclo infinito es un bucle que no se le asigna una condicion o parametro para que finalize por lo tanto se vuelve infinito y esto puede saturar nustra memoria y llegar a danar nuestro equipo

¿Puedo mezclar ciclos y condicionales?

Por Supuesto , En base a las condiciones que le demos a nuestro ciclo este podra funcionar de manera correcta

¿Qué es un array?

Una array es una lista de elementos que empienzo desde la posicion 0

¿Qué es un objeto?

Un objeto es un elemento que puede contener propiedades de este misma

¿Cuándo es mejor usar objetos o arrays?

Un array es recomendable usarlo cuando queramos hacer una lista de elementos , objetos y querramos ir almacenado mas elementos o ir jugando con estos y Un objeto es mejor usarlo para agregarle propiedades a un elemento

¿Puedo mezclar arrays con objetos o incluso objetos con arrays?

Si claro , Yo puedo tener un array de objetos y tambien puedo agregar un objeto a un array
